================================================================================
                         INSTALLATION ET UTILISATION
================================================================================

PRÉREQUIS:
-----------
1. Windows 10 ou 11
2. Visual Studio 2022 (Communauté, Professionnel ou Enterprise)
   → Avec la charge de travail "Développement .NET Desktop"
3. .NET 7.0 SDK (installé avec Visual Studio)

INSTALLATION:
-------------
1. Extraire ce fichier ZIP dans un dossier de votre choix
   Exemple: C:\Users\VotreNom\Documents\Projets\

2. Ouvrir Visual Studio 2022

3. Fichier > Ouvrir > Projet/Solution
   Naviguer vers: GestionEcole\GestionEcole.vbproj
   OU
   Naviguer vers: GestionEcole\GestionEcole.sln (si disponible)

4. Attendre que Visual Studio charge le projet
   → Les packages NuGet se restaureront automatiquement
   → Si ce n'est pas le cas: clic droit sur la solution > Restaurer les packages NuGet

5. Appuyer sur F5 ou cliquer sur le bouton "Démarrer" pour exécuter

================================================================================
                         FONCTIONNALITÉS PRINCIPALES
================================================================================

1. GESTION DES ÉTUDIANTS
   - Ajouter, modifier, supprimer des étudiants
   - Recherche multi-critères
   - Upload de photo de profil
   - Génération automatique des numéros d'inscription

2. GESTION DES RÉINSCRIPTIONS
   - Suivi annuel des niveaux (1ère à 5ème année)
   - Historique complet
   - Vérification des prérequis

3. GESTION DES PAIEMENTS
   - Enregistrement des paiements
   - Types de frais: inscription, scolarité, matériel, examen
   - Modes de paiement: espèces, virement, chèque
   - Suivi des statuts: complet, partiel, en attente
   - Génération de reçus PDF

4. TABLEAU DE BORD (NOUVEAU)
   - KPI visuels (total étudiants, paiements, montants)
   - Graphiques statistiques
   - Répartition par filière
   - Statut des paiements

5. RAPPORTS
   - Liste des étudiants filtrable
   - Liste des impayés
   - Export Excel
   - Export PDF

6. SÉCURITÉ
   - Authentification par login/mot de passe
   - Hashage SHA256 des mots de passe
   - Deux niveaux d'accès: Administrateur et Personnel
   - Journal de toutes les actions

================================================================================
                         RÉSOLUTION DES PROBLÈMES
================================================================================

PROBLÈME: "Les packages NuGet ne se restaurent pas"
SOLUTION:
  1. Outils > Gestionnaire de package NuGet > Console du gestionnaire de package
  2. Taper: Update-Package -reinstall
  3. Ou restaurer manuellement via: clic droit sur la solution > Restaurer les packages NuGet

PROBLÈME: "Erreur de compilation"
SOLUTION:
  1. Build > Nettoyer la solution
  2. Build > Reconstruire la solution
  3. Redémarrer Visual Studio

PROBLÈME: "Base de données introuvable"
SOLUTION:
  - La base de données se crée automatiquement au premier démarrage
  - Vérifier les droits d'écriture dans le dossier du projet
  - Vérifier que Microsoft Access Database Engine est installé

PROBLÈME: "L'application ne démarre pas"
SOLUTION:
  1. Vérifier que .NET 7.0 est installé
  2. Vérifier que Visual Studio est à jour
  3. Réinstaller les packages NuGet
