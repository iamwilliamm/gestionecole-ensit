================================================================================
                    INSTRUCTIONS POUR FINALISER
================================================================================

ÉTAPE 1: OUVRIR DANS VISUAL STUDIO
-----------------------------------
1. Ouvrir Visual Studio 2022
2. Fichier > Ouvrir > Projet/Solution
3. Sélectionner: C:\Users\hp\source\repos\GestionEcole\GestionEcole.vbproj
4. Attendre que les packages NuGet se restaurent

ÉTAPE 2: RESTAURER LES PACKAGES
-------------------------------
Si les packages ne se sont pas restaurés automatiquement:
1. Clic droit sur la solution > Restaurer les packages NuGet
2. Ou exécuter dans la console:
   dotnet restore

ÉTAPE 3: CORRIGER LES ERREURS EVENTUELLES
------------------------------------------
Si vous voyez des erreurs de compilation:

1. Erreurs sur LoginForm:
   - Supprimer le fichier: LoginForm.Designer.vb s'il existe encore
   - Le LoginForm.vb actuel est autonome (pas besoin de Designer)

2. Erreurs sur DashboardForm:
   - Le fichier DashboardForm.Designer.vb a été supprimé
   - Le DashboardForm.vb est autonome

3. Erreurs de types non définis:
   - Vérifier que les packages NuGet sont bien installés
   - Build > Nettoyer la solution
   - Build > Reconstruire la solution

ÉTAPE 4: CONFIGURER LE LOGO ENSIT
----------------------------------
Si vous avez un logo ENSIT en image:
1. Copier l'image dans: Resources\logo_ensit.png
2. Modifier ENSITTheme.vb pour charger l'image
3. Ou utiliser le logo dessiné en code (actuel)

ÉTAPE 5: TESTER L'APPLICATION
-----------------------------
1. F5 pour exécuter
2. Identifiants de test:
   - Admin: admin / 12345
   - Personnel: secretariat / 12345
3. Tester:
   - Connexion
   - Navigation dans les menus
   - Tableau de bord
   - Génération PDF (menu Paiements > Générer reçu)
